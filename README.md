# CryptoNexus
High-performance Real-Time Crypto Trading Terminal &amp; Market Data Aggregator. Built with the MERN Stack by Ifeanyi Nwambeke (Nwambeke Ifeanyi). Featuring live CoinGecko API integration, responsive Tailwind CSS UI, and AOS animations.


# ⚡ CryptoNexus Terminal
### Developed by [Ifeanyi Nwambeke](https://github.com/Nwambekeifeanyi)

**CryptoNexus** is a sophisticated, real-time cryptocurrency market monitoring tool designed for the decentralized web. This project serves as a core part of the professional portfolio of **Ifeanyi Nwambeke**, demonstrating expertise in full-stack web development, financial API integration, and high-performance UI/UX design.

---

## 🛠️ Tech Stack
* **Frontend:** HTML5
* **Styling:** Tailwind CSS (Emerald & Violet Theme)
* **Animations:** AOS (Animate On Scroll)
* **Data:** CoinGecko REST API
* **Icons:** Lucide-React
